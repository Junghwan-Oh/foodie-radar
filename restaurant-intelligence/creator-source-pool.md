# 맛집러/인플루언서 1차 소스 풀

Last updated: 2026-05-18

## 목적

이 문서는 개별 맛집 추천 목록이 아니라, 어떤 맛집러/인플루언서를 추적 소스로 삼을지 관리하는 1차 소스 풀이다.

단순 구독자수 순위표가 아니라 다음 목적을 가진다.

- 1군 맛집러를 고정 추적한다.
- 보조 맛집러는 카테고리별 교차검증 소스로 둔다.
- 현재 목적과 맞지 않거나 광고/협찬 리스크가 큰 채널은 보류 또는 제외한다.
- 새로 뜨는 맛집러는 바로 신뢰하지 않고 후보로 넣은 뒤 검증한다.
- 사용자가 직접 추가한 채널은 `*`로 표시한다.

## 운영 원칙

| 원칙 | 내용 |
|---|---|
| 1군 우선 | 1군 채널의 직접 방문/직접 추천 식당은 후보 큐에 우선 등록한다. |
| 직접 언급만 인정 | 스타일이 비슷하다는 이유로 추천 출처를 붙이지 않는다. |
| 광고 분리 | 유료광고, 제작지원, 초대, 협찬, 브랜드 캠페인은 맛 평가와 별도로 표시한다. |
| 취향 적합성 분리 | 소스 신뢰도와 사용자 취향 적합성은 별도 점수로 본다. |
| 무리한 Top 30 금지 | 숫자를 채우기 위해 아직 검증되지 않은 채널을 억지로 넣지 않는다. |
| 주기적 정리 | 추천 품질, 광고 패턴, 사용자 방문 후기에 따라 추가/수정/삭제한다. |

## 맛집러 선정 기준

1차 소스 풀은 단순 유명세나 조회수보다 추천의 질을 우선한다. 사용자가 선호하는 맛집러는 다음 둘 중 하나다.

- 사용자와 맛집 취향이 비슷한 사람
- 사용자보다 훨씬 고급/전문적인 취향을 가지고, 그 판단을 구체적으로 설명하는 사람

맛집러를 새로 추가할 때의 1차 필터:

| 기준 | 판정 |
|---|---|
| 나와 비슷한 맛집 취향 | 우선 검토. 실제 방문 후 적중률이 높으면 1군/보조로 승격 가능 |
| 나보다 훨씬 고급 맛집 취향 | 우선 검토. 파인다이닝/오마카세/장르 전문성은 보조 또는 장기 위시리스트로 활용 |
| 나와 취향도 다르고 고급 판단력도 아직 불명확 | 보류 또는 제외 |
| 홍보/마케팅 비중이 높고 음식 판단 근거가 부족함 | 보류 또는 광고 주의 |

선호하는 추천 신호:

- 인생맛집
- 해당 분야 국내 1위 / 장르 최고 / 이 분야 끝판왕
- 극강의 가성비이면서 품질이 실제로 높은 집
- 장인의 퀄리티가 드러나는 집
- 자주 가게 되는 단골 맛집
- 무조건 와봐야 할 경험
- 여러 후보를 비교한 뒤 남은 최종 추천

보류/강등 신호:

- 홍보, 마케팅, 바이럴 비중이 높고 음식 판단 근거가 부족함
- 광고/협찬이 많은데 비판이나 주문 디테일이 부족함
- 핫플 소개는 많지만 본인 취향의 기준이 흐림
- 식당 방문 판단보다 제품/브랜드 노출이 중심
- 사용자가 방문했을 때 반복적으로 취향이 빗나감

## 업데이트 규칙

1. 새 맛집러가 발견되면 바로 1군에 넣지 않고 `1차 후보 풀`에 추가한다.
2. 최근 1~3개월 콘텐츠에서 실제 식당 추천 비중, 광고 표기, 추천 사유의 구체성, 사용자 취향 적합성을 본다.
3. 사용자가 방문 후 만족하면 해당 맛집러의 취향 적합성을 올린다.
4. 숨은 광고 의심, 과한 홍보성 표현, 방문 후 불만족이 반복되면 보류 또는 제외한다.
5. 1군은 적게 유지한다. 현재 1군 6명도 충분히 많다.
