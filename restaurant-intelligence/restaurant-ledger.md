#### Course / Omakase Details

Required when the restaurant is course-based, tasting-menu, kappo, sushi omakase, omakase, or chef's counter.

| Course/Menu | Price | Key Dishes | Dish-Level Features | Source/Date | Notes |
|---|---:|---|---|---|---|
|  |  |  |  |  |  |

Rules:
- Do not stop at "course meal" or "omakase" as a category label.
- List the main course items or representative sequence when available.
- For each important dish, note what makes it distinct: ingredient, cut, aging, broth/sauce, rice/shari, cooking method, texture, balance, portion, or seasonal feature.
- If only a partial course list is available, mark it as partial instead of implying completeness.
- If course composition changes seasonally or daily, state that clearly and keep the date of the source.
