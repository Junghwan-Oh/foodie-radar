# Creator Lifecycle

Last updated: 2026-05-18

## Purpose

Track creator additions, status changes, trust upgrades/downgrades, archival, and deletion. A creator being "hot" is a discovery signal, not a trust signal.

## Status Definitions

| Status | Meaning |
|---|---|
| hot-candidate | Newly hot or repeatedly surfaced; not audited. |
| watch | Worth watching but not trusted yet. |
| active | Audited and useful; direct mentions may be used with trust score. |
| probation | Trust weakened; require corroboration. |
| inactive | Not producing useful current food content. |
| archived | Historical record kept but not actively scanned. |
| excluded | Do not use for recommendations except when explicitly requested. |
